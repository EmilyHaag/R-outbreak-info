getSeqCounts

Description:

Returns a dataframe of number of sequences per day by location

Usage:

getSeqCounts(location=NULL, cumulative=NULL, subadmin=NULL)

Arguments:

location: a location name (if not specified, global total counts returned)

cumulative: Boolean (T/F), T returns cumulative number of sequences to date

subadmin: Boolean (T/F), subadmin=T and cumulative=T returns cumulative number of sequences for next administrative level

df=getSeqCounts(location="United States of America")

getAllLineagesByLoc

Description:

Returns prevalence of all lineages over time for a specified location

Usage:

getAllLineagesByLoc(location, other_threshold=0.05, nday_threshold=10, ndays=180, other_exclude=NULL, cumulative=F)

Arguments:

location: a location name

other_threshold: minimum prevalence threshold below which lineages are accumulated under “Other”

nday_threshold: minimum number of days in which the prevalence of a lineage must be below other_threshold to be accumulated under “Other”

ndays: the number of days before the current date to be used as a window to accumulate lineages under “Other”

other_exclude: comma separated lineages that are NOT to be included under “Other” even if the conditions specified by the three thresholds above are met

cumulative: Boolean (T/F), T returns cumulative prevalence of lineages

Example:

df=getAllLineagesByLoc(location = "India", other_threshold=0.03, nday_threshold=60, other_exclude="P.1")
print(head(df))
